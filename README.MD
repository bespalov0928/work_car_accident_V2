### О проекте
Проект - Автонарушители
В системе существуют две роли (пользователь, автоинспектор).
Пользователь создает заявление правонарушения.
В заявлении указывает: адрес, номер машины, описание нарушения.

### Стек технологий

* Spring Boot
* PostgreSQL
* Liquibase

* jacoco
